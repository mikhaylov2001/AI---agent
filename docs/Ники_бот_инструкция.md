# Ники — личный наставник (инструкция)

Интеграция из `Ники_бот_инструкция.docx` в текущий Spring Boot бот v2.0.4.

## Что реализовано

| Из docx | В боте |
|---------|--------|
| Claude API | `LLM_PROVIDER=claude`, `CLAUDE_API_KEY`, `AnthropicConfig` |
| Память диалога | PostgreSQL `ChatMessage` + `memorySummary` (не in-memory) |
| Утро 8:00 / вечер 21:00 MSK | cron 05:00 / 18:00 UTC + `/autopilot on` |
| Только владелец | `TELEGRAM_OWNER_ID` — автопилот только ему |
| 4 цели Дмитрия | `mentor/instructions.txt` + профиль по умолчанию |
| `/цели` | Показывает 4 цели + цели из бота |
| `/сброс` | Очищает историю чата и долгосрочную память |
| Стиль без воды | Промпт + `MentorResponseFormatter` |

## Переменные на Render

```env
LLM_PROVIDER=claude
CLAUDE_API_KEY=sk-ant-...
CLAUDE_MODEL=claude-sonnet-4-6
TELEGRAM_OWNER_ID=123456789
TELEGRAM_BOT_TOKEN=...
TELEGRAM_DELIVERY_MODE=webhook
TELEGRAM_WEBHOOK_URL=https://ai-agent-fgo1.onrender.com
NIKI_CRON_SECRET=...
DB_URL=...
```

Groq (запасной вариант): `LLM_PROVIDER=groq` + `GROQ_API_KEY`.

## Команды

- `/start` — приветствие и меню
- `/цели` — четыре главные цели
- `/сброс` — сброс памяти диалога
- `/autopilot on` — утренний план и вечерний чек-ин

## VPN

Anthropic API из РФ может требовать VPN. Groq работает без VPN.

## Модель

По умолчанию: `claude-sonnet-4-6`.
Для Opus: `CLAUDE_MODEL=claude-opus-4-6`.
